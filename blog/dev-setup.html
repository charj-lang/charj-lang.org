Charj Dev Setup

macOS

  1. install LLVM
brew install llvm
  1. build in first time
LLVM_SYS_110_PREFIX=/usr/local/Cellar/llvm/11.0.0 cargo build

Ubuntu

  1. install LLVM
wget https://apt.llvm.org/llvm.sh && chmod +x llvm.sh && sudo ./llvm.sh 10
  1. build
LLVM_SYS_110_PREFIX=/usr/local/Cellar/llvm/11.0.0 cargo build
